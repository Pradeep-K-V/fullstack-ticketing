## Overview
This repository contains a ready-to-run starter MERN ticketing system:
- Backend: Node.js, Express, MongoDB, JWT validation via Auth0 JWKS
- Frontend: React + Vite

## Structure
- backend/ : Express API
- frontend/: Vite + React SPA

See `.env.example` files in each folder and replace placeholders before running.

## How to run (locally)
1. Backend:
   - cd backend
   - cp .env.example .env   (fill values)
   - npm install
   - npm run dev
2. Frontend:
   - cd frontend
   - cp .env.example .env   (fill values)
   - npm install
   - npm run dev
